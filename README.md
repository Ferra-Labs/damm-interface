# Ferra DAMM Interface Documentation

This repository contains the Move interface definitions for the Ferra Concentrated Liquidity Market Maker (DAMM) protocol. These interfaces provide type definitions and function signatures without implementations, allowing developers to interact with Ferra DAMM without needing the full implementation.

## Overview

The Ferra DAMM interface consists of three main modules:

1. **`factory.move`** - Factory interface for creating and managing liquidity pools
2. **`pool.move`** - Pool management interface (core trading, swap operations, and liquidity management)
3. **`position.move`** - Position management interface (user position tracking, fee collection, and liquidity provision)
